\# PowerShell Automation



Enterprise-grade PowerShell automation tooling.



\## 📦 Projects



\- Certerator – Certificate discovery \& renewal automation

\- ADorator – Active Directory auditing and analysis

\- DCUpgraderator – Domain Controller lifecycle automation



\## 🧠 Design Principles



\- Fully portable scripts (no external dependencies)

\- Inline asset embedding (Base64)

\- Strong logging + auditability

\- Idempotent execution

\- Enterprise-safe (no destructive defaults)



\## 📁 Structure



\- Modules/     → Shared reusable functions

\- Scripts/     → Primary tools

\- Assets/      → Images, Base64 data

\- Docs/        → Documentation + usage

\- Tests/       → Validation and test logic
